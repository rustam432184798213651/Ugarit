ВАЖНО! Комрьютер и телефон должны быть подключены к одной и той же интернет сети. ВАЖНО!
На вашем компьютере:
Разархивируйте архив в папку на ваш выбор.
Чтобы запустить мобильное приложение "eBook" необходимо:

Установить Node.js и настроить переменные глобальной среды. Чтобы сделать это необходимо следовать инструкции, которая есть на сайте: https://www.interserver.net/tips/kb/install-configure-nodejs-npm-windows/#:~:text=To%20install%20Node.js%20and%20npm,easily%20run%20from%20any%20directory

Нужно установить VScode, если его нет. Следуйте инструкции на видео: https://www.youtube.com/watch?v=MAuLRELi2YA
После установки VScode зайдите в папку, где лежат разархивированные файлы. Откройте терминал в данной директории, это можно сделать, нажав правой кнопкой мыши по свободному месту в директории и нажать "Open in terminal". Напишите "code ." и прожмите "Enter", в открывшемся терминале. В результате появится Visual Studio Code со всеми файлами и директориями. Далее нажмите на "..." -> "Terminal" -> "New Terminal". Откроется терминал VScode.

Далее я перечислю список команд. Стоит отметить, что каждая команда занимает время на отработку. Нужно ждать пока текущая команда полностью отработает, и только после этого следует запускать следующую. Чтобы запустить команду в VScode терминале достаточно ее написать и прожать "Enter". В открывшемся терминале запускаем следующие команды: 
npm install -g npx
npm i expo
npm i react
npm i react-native
npm i @react-navigation/native
npm i @react-navigation/native-stack
npm i expo-sharing
npm i expo-file-system
npm i expo-document-picker
npm i expo-intent-launcher
npm i @bildau/rn-pdf-reader
npm i react-native-popup-menu


Далее, введите команду npx expo start в терминале VScode.

Теперь перейдем к телефону, с которого вы хотите запустить приложение.

На вашем телефоне:
Если вы хотите запустить приложение на IOS, то зайдите в "AppStore" и установите "Expo Go". Далее, запустите приложение "Camera" и наведите камеру на штрих код, который находится в терминале VScode. Появится prompt: "Open in Expo Go". Нажмите на него. На этом запуск приложения окончен.
Если же вы хотите запустить приложение на Android, то зайдите в "Google Play" и установите "Expo". Откройте "Expo" и нажмите "Scan QR code", после чего наведите на QR code камеру. На этом запуск приложения окончен.
